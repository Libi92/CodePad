Virtual lab

At the time of learning languages like C, C++, Java Etc. The beginners had number of problem like to install compilers at desktop-pc. Some time student had to face the problem of configuring the compiler. So that they can’t get execute the program successfully. And other problem is that if the users want to compile program on any other system they require again all processing to install and configuration problem.

To solve this problem a python application is designed to execute the program on different types of source code by one editor. And also a well configured compiler providing services by servers.Simply to create a common server for different programming language.To create a common interface for all programming language..
